### DevTrack

A lightweight study app for practicing and tracking knowledge through question-based learning, similar to flashcards but with progress tracking and randomized test sessions.



#### Features



* Add and organize study questions by category and difficulty
* Track progress (Not started, Practicing, Mastered)
* Search and filter your question bank
* Persist data locally using localStorage
* Generate randomized study sessions from saved questions
* View basic progress stats





#### Built With



* HTML
* CSS
* JavaScript
* Browser localStorage





#### Purpose



I built this project as a flexible study tool that combines the simplicity of flashcards with structured tracking and testing. It also helped me strengthen my frontend JavaScript skills, especially working with DOM manipulation, state management, and persistent data.
